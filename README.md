## Features

- **Fast Compilation:** Optimized compiler for quick build times.
- **Simple Syntax:** Readable and developer-friendly syntax.
- **Native Mobile Support:** Directly targets iOS and Android.
- **Powerful Debugging Tools:** Integrated debugging for efficient development.
- **VS Code Extension:** Syntax highlighting, linting, auto-completion, and debugging support.
